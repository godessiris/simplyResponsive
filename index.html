@media queries

CSS uses the @media rule to detect information about the device being used to access the website, for example it can be used to detect. The size of the device it's capavilities and its orientation. For this course wil be using to detect the device size, but keep in mind that is capable of much more.

Phone: Width less than or equal to 600.

Tablet: Width between 601 and 1023.

Desktop: Width greater than or equal to 1024.
